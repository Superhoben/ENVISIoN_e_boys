About

ENVISIoN is an open source tool/toolkit for electron visualization developed at Linköping University.

ENVISIoN is implemented using (a modified version of) the Inviwo visualization framework, developed at the Scientific Visualization Group at Linköping University (LiU).

For more information see the github repository at: https://github.com/rartino/ENVISIoN
